# 先天八卦64卦立方体模型——快速对齐文档

## 一、底层定义（最重要，不可偏移）

**0 = 阳 = 无 = 空 = 未显**
**1 = 阴 = 有 = 色 = 已显**

这是与传统易学的根本性颠覆，同时在数学、物理、哲学三个层面成立：
- 数学：0是基态，1是激发态
- 物理：真空（0）是所有可能性的母体，粒子（1）是从真空中显现的激发
- 哲学：老子的"道"对应0（无），万物对应1（有）

乾(000) = 纯粹未显化的可能性空间，不是"天的刚健"
坤(111) = 完全显化的物质世界，不是"地的柔顺"
从乾到坤 = 宇宙从无到有的创生方向

## 三、64卦的完整生成逻辑

**8个顶点 = 8个纯卦（上下卦相同，内部能量自振/驻波）**

**28条连线 × 2方向 = 56个杂卦（能量在两顶点间双向流动）**
- 12条棱线（汉明距离1，单爻变）
- 12条面对角线（汉明距离2，双爻变）
- 4条体对角线（汉明距离3，三爻全变，即错卦）

**8 + 56 = 64，完备，无遗漏，无重复**

能量方向决定取卦：从哪个顶点出发流向哪个顶点，决定上卦（主场）和下卦（客场）。

## 四、动力学机制

**能级小 → 纯卦驻波振荡**（系统在本征态内，不跃迁）

**能级大 → 突破边界形成杂卦场**（系统激发，跃迁）
- 汉明距离1（棱线）：最小能量门槛，单爻变
- 汉明距离2（面对角线）：中等能量，双爻变
- 汉明距离3（体对角线）：最大能量，三爻全变

几何距离 = 能量门槛 = 爻变数量，三者是同一件事。

6爻不是静态符号堆叠，而是**时间扫描的截面**——能量沿连线流动过程中，时间轴在每个瞬间切出的横截面。
